# Full-Fledged-Ls-implementation

## Commands 
```
$ gcc myls.c -o myls
$ ./myls
$ ./myls -i
$ ./myls -i -a
$ ./myls -i -a -l
$ ./myls . / -l 
```

## Features
- Support options like -i -a -l
- Support Listing one or multiple directories 
- Support long format (file type, permissions, num of links, user, group, time, ...)
- Show or hide hidden files
